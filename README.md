# Gamified Learning

> **Live site:** [gamifiedlearning.org](https://gamifiedlearning.org)

Welcome to **Gamified Learning** — an interactive, story-driven platform that
makes education engaging through puzzles, narratives, and game-based challenges.

## Featured Adventure

### The Mystery of the Seven Stars

An immersive learning experience where players explore an unfolding
mystery while developing critical thinking, problem-solving, and
subject-matter knowledge.

- 📖 [Read / Download the Adventure](assets/mystery-of-the-seven-stars.pdf)
- 🖼️ [Cover Art](assets/cover.png)
